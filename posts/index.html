    Hack the Box - Traceback

    Traceback is a simple Linux box. I gain access to it using an already existing webshell as a low-privileged user. I write a script to gain access as another user, and discover that the motd file is writable. I add a reverse shell to it, allowing me to gain root access when a user logs in via SSH.
